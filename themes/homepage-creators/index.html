ℹ️ 简介

Homepage Creators 是一款适用于 Zola 的个人主页主题，风格接近 Apple，美观大气。

注意：本主题移植于的开源 HeoWeb 纯静态主题，感谢 张洪 Heo 的无私分享。

本主题使用简单，只需修改config.toml文件即可动态调整内容。无需像上游仓库一样修改 HTML 内容，极大降低用户的心智负担。

已完成所有功能移植

📝 快速上手

  1. 参考官方说明安装 Zola 命令行工具

  2. 将本仓库克隆到本地

    git clone --depth=1 https://github.com/iWangJiaxiang/Homepage-Creators

  3. 进入本地仓库

    cd zola-theme-jiaxiang

  4. 运行预览命令，然后在浏览器打开提示的预览地址

    zola serve

    此时您应该成功访问到博客网站啦

  5. 参考官方资料进一步并定制您的主页

    • zola 命令说明
    • 理解项目结构
    • 自定义

  6. 根据需要修改config.toml配置文件内容，您需要对 TOML 格式有基本的了解。

  7. 根据需要将自己的图片素材放到static/img文件夹
